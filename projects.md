# Projects

## Hansard NLP Explorer

Socio-technical analysis of the Hansard Debate Corpus from the UK Parliament, spanning over 200 years of parliamentary discourse (1803-2005).

**Scale:** 1.2M debates, 6.0M speeches

**Key Analysis:** Women's Suffrage Classification
- Classified 2,808 speeches with 92.9% accuracy using LLM-based methods
- Extracted 5,138 arguments across 9 taxonomy categories
- Historical findings: Documented the 95% drop in suffrage-related speeches from 1913-1915 coinciding with WWI

**Technical approach:**
- Large-scale text processing with pandas and polars
- LLM-powered argument classification and extraction
- Gender analysis with temporal validation across MPs
